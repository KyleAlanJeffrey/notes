"*[Advanced Package Tool (or APT)](https://en.wikipedia.org/wiki/APT_%28Package_Manager%29 "WikiPedia"), the main command-line [package manager](https://wiki.debian.org/PackageManagement) for Debian and its derivatives. It provides command-line [tools](https://wiki.debian.org/PackageManagementTools) for searching, [managing](https://wiki.debian.org/PackageManagement) and querying information about packages, as well as low-level access to all features provided by the libapt-pkg and libapt-inst libraries which higher-level package managers can depend upon.*"
## Helpful Commands

| Ubuntu                     | Params        | Description                    |
| -------------------------- | ------------- | ------------------------------ |
| `apt list`                 | `--installed` | Show all packages              |
| `apt-cache search keyword` |               | Show available package keywrod |
| `dpkg -i package`          |               | Install debian                 |
| `dpkg -I package`          |               | View Debian package version    |
