*dpkg in Linux is the **primary package manager for Debian and Debian-based systems  The tool installs, builds, removes, configures, and retrieves information for Debian packages. The command works with packages in . deb format.***

**[[APT]]** sits on top of dpkg.

**[dpkg-buildpackage](https://man7.org/linux/man-pages/man1/dpkg-buildpackage.1.html)** - A program that automates building debian packages.
